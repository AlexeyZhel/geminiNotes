ANGULAR GEMINI TEST APPLICATION

Requirements:
node >=0.10.0

Installation:

Install gulp and bower: 
- npm install -g bower gulp gulp-cli

Install applcitaion requirements using npm: 
- npm install

Install application requirements using bower:
- bower install

Build and run application:
- gulp serve:dist

To run unit tests:
- gulp test

To run e2e tests:
- gulp protractor
